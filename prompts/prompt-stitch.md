# prompt-stitch.md

Objectif : produire un design mobile-first premium, spa haut de gamme, minimaliste, très lisible.

## Entrées
- Marque : La Belle Détente
- Univers : luxe épuré, blanc cassé, beiges, typographies élégantes
- CTA : Réserver un massage (principal), Collaborer (secondaire)
- Sections : hero, preuves premium, services, professionnels, FAQ, contact

## Références
- Style "spa luxe", beaucoup d’espace, micro animations sobres
- Pas de surchargé, pas de carrousels lourds

## Contraintes
- Navigation sticky mobile
- CTA visibles sans scroll
- Accessibilité AA, contrastes suffisants

## Sorties attendues
- 3 écrans mobiles : Accueil, Prestations, Professionnels
- Design tokens : tailles, typos, spacing, radius, buttons
